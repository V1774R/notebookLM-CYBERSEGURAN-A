## Fundamentos e Princípios
A base da segurança da informação repousa sobre a **Tríade CIA**:
- **Confidencialidade**: Garantir que os dados sejam acessíveis apenas por pessoas autorizadas.  
- **Integridade**: Manter a consistência, precisão e confiabilidade dos dados contra alterações não autorizadas.  
- **Disponibilidade**: Assegurar que sistemas e dados estejam prontos para uso quando necessário.  

Além disso, o material cobre modelos de rede essenciais como **OSI (7 camadas)** e **TCP/IP (4 camadas)**, fundamentais para entender a comunicação digital.

## Ameaças e Ataques Cibernéticos
Estudo detalhado do funcionamento e mitigação das principais ameaças modernas:
- **Malware**: Vírus, Worms, Trojans e Ransomware.  
- **Engenharia Social**: Phishing, Spear Phishing e Vishing.  
- **Ataques de Injeção**: SQL Injection e Cross-Site Scripting (XSS).  
- **DDoS (Negação de Serviço Distribuída)**: Uso de botnets para sobrecarregar servidores.  
- **Man-in-the-Middle (MitM)**: Interceptação de comunicações entre duas partes.

## Metodologia de Hacking Ético
O hacking ético envolve pensar como um invasor para fortalecer as defesas, seguindo seis fases críticas:

1. **Reconhecimento (Footprinting)**: Coleta de informações sobre o alvo.  
2. **Varredura (Scanning)**: Identificação de portas abertas e serviços vulneráveis.  
3. **Ganho de Acesso (Exploitation)**: Exploração ativa das vulnerabilidades.  
4. **Manutenção de Acesso**: Instalação de backdoors para acessos futuros.  
5. **Limpeza de Rastros**: Remoção de evidências da intrusão.  
6. **Relatório**: Documentação completa das falhas e recomendações de correção.

## Criptografia e Proteção de Dados
Exploração dos mecanismos para tornar dados ininteligíveis a terceiros:
- **Criptografia Simétrica**: Mesma chave para cifrar e decifrar (AES, DES).  
- **Criptografia Assimétrica**: Par de chaves pública/privada (RSA).  
- **Hashing**: Funções de via única para integridade (MD5, SHA-256).  
- **Protocolos de Segurança**: SSL/TLS para navegação segura em sites HTTPS.
